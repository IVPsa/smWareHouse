How to choose the proper License

Depending on what kind of applications you are developing and what OS your client requires, you can choose different licenses. For more information, please refer to the License Agreement.

If you are developing a desktop application

You need

  • DEVELOPER LICENSE: on a per-Application per-named programmer basis.
  • SITE LICENSE: allows unlimited amount of developers at a single geographic location to work on the barcode module.

Please NOTE that

  1. It is applicable to Windows Edition and Mac Edition.
  2. The edition means which end user OS is supported.
  3. Royalty free for desktop application deployment.

If you are developing a web application

You need

  • PRODUCTION SERVER LICENSE: one license per Application per Server.
  • Non-PRODUCTION SERVER LICENSE: one license per Application per Server.
  • UNLIMITED-SERVER YEARLY LICENSE: licensed on a per Application per year basis. Allows you to deploy one Application to an unlimited number of Servers within the coverage period.

Please NOTE that

  1. It is applicable to Windows Edition and JavaScript Edition.
  2. Development servers, testing servers, and staging servers are all considered non-production servers.
  3. Currently there is no limit as to how many desktops can access the software once it's deployed to the production server.

If you are developing a mobile application

You need

  • DEVELOPER LICENSE: one license per developer who needs to work on the barcode related code.

Please NOTE that

  1. It is limited to one developed App.
  2. It covers both Android and iOS Apps.
  3. Allows single App distribution on unlimited devices through Android's Google play store or Apple's iTunes store.
