Deployment

For server side operation system, Dynamsoft Barcode Reader JavaScript SDK supports Windows, Linux, Unix, Mac, etc. For more info on the basic requirements, please click here.

Generally speaking, there is no restriction on the operation system. Deployment rules apply to all types of server machines.

All you need to do is to deploy your application as well as our product (DBRResources folder which includes exe installer, JavaScript files, etc.) to your server, then when your clients go to your webpage, they will be prompted to install Dynamsoft Service and Barcode Reader components. After installing them, they will be able to read barcodes in the browser.

Typically, the DBRResources folder can be found under C:\Program Files (x86)\Dynamsoft\Barcode Reader JavaScript SDK {Version Number}\.
