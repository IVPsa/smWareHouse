readBinaryAsync Method

Description

Reads barcodes from a blob image asynchronously. It supports jpg, png, tif, gif and bmp formats. This method is valid only in IE 10-11, Edge, Chrome v27+ and Firefox v27+.

Syntax

.readBinaryAsync( Blob blobImage, 
                  Anything userData, 
                  OnBarcodeReadSuccess asyncSuccessFunc, 
                  OnBarcodeReadFailure asyncFailureFunc);

Data type

Void

Parameters

blobImage
The binary array of the barcode image.
userData
Custom user data, such as image index, image url, etc.
OnBarcodeReadSuccess asyncSuccessFunc
Callback function triggered when barcode reading succeeds.
OnBarcodeReadFailure asyncFailureFunc
Callback function triggered if barcode reading fails.

Example

// HTML:
<input type="file" id="file"/>

// JavaScript:
var upLoadFile = document.getElementById("file").files[0];
dbrObject.readBinaryAsync(file, "BarcodeImage.bmp", OnBarcodeReadSuccess, OnBarcodeReadFailure);

function OnBarcodeReadSuccess(userData, result){
    var count = result.getCount();
    var strMsg = "";
    if(count > 0) {
        for (var i = 0; i < count; i++){
            strMsg +="Index: " + i + "\n";
            strMsg +="Barcode Type: " + result.get(i).formatString + "\n";
            strMsg +="Barcode Value: " + result.get(i).text + "\n";
        }
        alert(strMsg);
    }
    else {
        alert("No barcode(s) found.");
    }
}
function OnBarcodeReadFailure(userData, errorCode, errorString){
	alert(errorCode + errorString);
}
