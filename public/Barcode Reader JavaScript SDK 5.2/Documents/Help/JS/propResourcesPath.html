dynamsoft.dbrEnv.resourcesPath Property

Description

Specifies where the dependencies are placed. This is a relative path to the current web page.

Data Type

String

Syntax

dynamsoft.dbrEnv.resourcesPath = "...";

Remarks

The path is very important as it points to all required JavaScript files, CSS files, images, etc. for the SDK to work correctly.

Example

// Set it in the dynamsoft.barcodereader.config.js file:
dynamsoft.dbrEnv.resourcesPath = "DBRResources";
