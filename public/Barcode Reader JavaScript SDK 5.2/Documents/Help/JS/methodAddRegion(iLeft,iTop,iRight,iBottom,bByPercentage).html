addRegion Method

Description

Adds an area of the image to the selection for barcode recognition.

Syntax

.addRegion(int iLeft, int iTop, int iRight, int iBottom, Boolean bByPercentage);

Parameters

iLeft
The left-most coordinate of the area.
iTop
The top-most coordinate of the area.
iRight
The right-most coordinate of the area.
iBottom
The bottom-most coordinate of the area.
bByPercentage
Indicates whether the coordinate is measured by percentage or not.

Remarks

By default, the barcode reader will search the whole image. You can speed up the recognition process by specifying a search area.

If bByPercentage is set to true, the allowed values for iLeft, iTop, iRight, iBottom range from 0 to 100. If it is set to false, the coordinates are measured in pixels.

If iRight <= iLeft or iBottom <= iTop or any of them < 0, it will return CustomedRegionInvalid.

Return value

bool

true indicates success. false indicates failure.

When false is returned, check getErrorCode or getErrorString for error information.

Example

var dbrObject;
dynamsoft.dbrEnv.init(function () { dbrObject = new dynamsoft.dbrEnv.BarcodeReader(); }, function (errCode, errMsg) { alert(errMsg); });

dbrObject.clearAllRegions();
dbrObject.addRegion(0, 100, 100, 120, false);
dbrObject.addRegion(200, 100, 300, 120, false);
