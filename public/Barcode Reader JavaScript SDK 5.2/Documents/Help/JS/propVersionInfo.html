dynamsoft.dbrEnv.versionInfo Property

Description

Returns the version number of the Dynamsoft Barcode Reader JavaScript file and the module dll. This is a read-only property.

Syntax

dynamsoft.dbrEnv.versionInfo;

Return value

String: "JS version: 5.2.0.0829, Dll version: 5.2.0.0829"

Remarks

"JS version" returns the version number of DBR JavaScript file (dynamsoft.barcodereader.initiate.js).

"Dll version" returns the version number of the Dynamsoft Barcode Reader module dll (dbr_{version number}.dll which is under C:\Windows\SysWOW64\Dynamsoft\DynamsoftService or C:\Windows\System32\Dynamsoft\DynamsoftService).

If you haven't installed Dynamsoft Barcode Reader module or the object is not initialized, "Dll version" will return an empty string.
