BarcodeReader Class

Description

Defines a class that provides functions for working with extracting barcode data.

Properties

  • Name Description
    BarcodeReader::barcodeColorMode Gets or sets the ink color for barcodes search.
    BarcodeReader::barcodeFormats Gets or sets the type(s) of barcodes to read.
    BarcodeReader::barcodeTextEncoding Gets or sets barcode text encoding mode.
    BarcodeReader::imageCaptureDevice Gets or sets the type of devices you used to capture barcode images.
    BarcodeReader::maxBarcodesNumPerPage Gets or sets the maximum number of barcodes allowed to be read per page.
    BarcodeReader::timeoutPerPage Gets or sets the maximum amount of time Dynamsoft Barcode Reader SDK should spend searching for a barcode per page.
    BarcodeReader::useOneDDeblur Gets or sets whether to deblur 1D barcodes before scanning.

Methods

  • Name Description
    Decode Barcode Methods
    BarcodeReader::readURL Reads barcodes from an image URL synchronously. It supports jpg, png, tif, gif and bmp formats.
    BarcodeReader::readURLAsync Reads barcodes from an image URL asynchronously. It supports jpg, png, tif, gif and bmp formats.
    BarcodeReader::readBase64 Reads barcodes from an image encoded as a base64 string synchronously.
    BarcodeReader::readBase64Async Reads barcodes from an image encoded as a base64 string asynchronously.
    BarcodeReader::readBinary Reads barcodes from a binary image synchronously.
    BarcodeReader::readBinaryAsync Reads barcodes from a binary image asynchronously.
    Get Error Methods
    BarcodeReader::getErrorCode Returns the error code.
    BarcodeReader::getErrorString Returns the error string.
    Set Page Methods
    BarcodeReader::addPages Adds page indices of a file (.tiff) to the selection for the barcode search.
    BarcodeReader::clearAllPages Clears all pages from the selection.
    Set Barcode Region Methods
    BarcodeReader::AddRegion (int iLeft, int iTop, int iRight, int iBottom, bool bByPercentage) Adds an area of the image to the selection for barcode recognition.
    BarcodeReader::AddRegion (ScanRegionSide eSide, int iPercentage) Adds an area of the image to the selection for barcode recognition.
    BarcodeReader::clearAllRegions Clears all regions from the selection.
    Set Barcode Angle Methods
    BarcodeReader::addAngleRange Adds a range of angles (in degrees) to the selection for the barcode search.
    BarcodeReader::addAngle Adds an orientation type to the selection for the barcode search.
    BarcodeReader::clearAllAngles Clears all angle ranges from the selection.
