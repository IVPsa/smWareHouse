EnumImageCaptureDevice Enumeration

Description

An enumeration that represents the type of devices you used to capture barcode images.

  • Member Description
    EICD_Unknown (Default) The image capture device type is unknown, try EICD_Scanner first and if no barcodes were found, try EICD_Fax and then EICD_Camera.
    EICD_Scanner Barcode images are captured by scanners.
    EICD_Camera Barcode images are captured from cameras or video devices.
    EICD_Fax Barcode images are captured from faxes.

Remarks

This ImageCaptureDevice enumeration is used as the type for the BarcodeReader.imageCaptureDevice property and is used to determine the type of devices when reading barcodes.

Barcode images captured by scanners may have salt-and-pepper noise. They are usually in grayscale or B&W.

Barcode images captured by cameras may suffer problems different than the ones captured by scanners, e.g., shadows, flash reflection, have poor perspective. They are usually in color.

Barcode images captured by faxes are usually in B&W.

EICD_Unknown will try EICD_Scanner type first and then if no barcodes were found, it will try EICD_Fax and then EICD_Camera. Therefore, using Unknown may naturally result in slower recognition speed. If you are not sure about the type of devices, please use EICD_Unknown.
