timeoutPerPage Property

Description

Gets or sets the expected maximum amount of time Dynamsoft Barcode Reader SDK should spend searching for a barcode per page.

Syntax

.timeoutPerPage = ...;

Data type

Int

Default Value

0x7fffffff

Remarks

It does not include the time taken to load/decode an image (Tiff, PNG, etc) from disk into memory.

This property is an expected value. If one of the barcode reading steps is still running when the specified timeout period has elapsed, the library will wait for the step to complete, so the time spent on barcode reading might be more than the given value.

Example

var dbrObject;
dynamsoft.dbrEnv.init(function () { dbrObject = new dynamsoft.dbrEnv.BarcodeReader(); }, function (errCode, errMsg) { alert(errMsg); });

dbrObject.timeoutPerPage = 10000;
