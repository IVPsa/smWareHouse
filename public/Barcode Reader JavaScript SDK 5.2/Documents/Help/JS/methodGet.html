get Method

Description

Returns a barcode at a given index of an array which represents all detected barcodes including format, contents, boundary, angle, etc.

Syntax

.get(int index);

Parameters

index
The index of the barcode result array. It is 0-based.

Return value

ReadResult

Methods available in ReadResult object:

Name Description
.get(index).format Returns the format of a barcode.
.get(index).formatString Returns the format string of a barcode.
.get(index).text Returns the barcode text as a string.
.get(index).data Returns the barcode data as an array of bytes.
.get(index).angle Returns the angle of a barcode. Values range from 0 to 360.
.get(index).moduleSize Returns the barcode module size (the minimum bar width in pixel).
.get(index).page Returns the page number the barcode is located on. The index is 0-based.
.get(index).left The left-most coordinate of the barcode.
.get(index).top The top-most coordinate of the barcode.
.get(index).width The width of the barcode.
.get(index).height The height of the barcode.
.get(index).x1 Returns the X coordinate of the top-left point of the barcode.
.get(index).y1 Returns the Y coordinate of top-left point of the barcode.
.get(index).x2 Returns the X coordinate of top-right point of the barcode.
.get(index).y2 Returns the Y coordinate of top-right point of the barcode.
.get(index).x3 Returns the X coordinate of bottom-right point of the barcode.
.get(index).y3 Returns the Y coordinate of bottom-right point of the barcode.
.get(index).x4 Returns the X coordinate of bottom-left of the barcode.
.get(index).y4 Returns the Y coordinate of bottom-left of the barcode.

Example

var result = dbrObject.readURL("http://demo1.dynamsoft.com/dbr/code128.png");
// Show barcode results
var count = result.getCount();
var strMsg = "";
if(count > 0) {
    for (var i = 0; i < count; i++){
        strMsg +="Index: " + i + "\n";
        strMsg +="Barcode Type: " + result.get(i).formatString + "\n";
        strMsg +="Barcode Value: " + result.get(i).text + "\n";
        strMsg +="Barcode Angle: " + result.get(i).angle + "\n";
    }
    alert(strMsg);
}
else {
    alert("No barcode(s) found.");
}
