ReadResult Class

Description

A class that returns all detected barcode results including format, contents, boundary, angle, etc.

Public Members

Methods

  • Name Description
    ReadResult::getCount Gets the count of all detected barcodes.
    ReadResult::get Gets a barcode by the index of an array which represents all detected barcodes including format, contents, boundary, angle, etc.

Properties

  • Name Description
    ReadResult::get(index).format Returns the format of a barcode.
    ReadResult::get(index).formatString Returns the format string of a barcode.
    ReadResult::get(index).text Returns the barcode text as a string.
    ReadResult::get(index).data Returns the barcode data as an array of bytes.
    ReadResult::get(index).angle Returns the angle of a barcode. Values range from 0 to 360.
    ReadResult::get(index).moduleSize Returns the barcode module size (the minimum bar width in pixel).
    ReadResult::get(index).page Returns the page number the barcode is located on. The index is 0-based.
    ReadResult::get(index).left The left-most coordinate of the barcode.
    ReadResult::get(index).top The top-most coordinate of the barcode.
    ReadResult::get(index).width The width of the bounding rectangle of the barcode.
    ReadResult::get(index).height The height of the bounding rectangle of the barcode.
    ReadResult::get(index).x1 Returns the X coordinate of the left-most point of the barcode.
    ReadResult::get(index).y1 Returns the Y coordinate of the left-most point of the barcode.
    ReadResult::get(index).x2 Returns the X coordinate of (x2, y2).
    ReadResult::get(index).y2 Returns the Y coordinate of (x2, y2).
    ReadResult::get(index).x3 Returns the X coordinate of (x3, y3).
    ReadResult::get(index).y3 Returns the Y coordinate of (x3, y3).
    ReadResult::get(index).x4 Returns the X coordinate of (x4, y4).
    ReadResult::get(index).y4 Returns the Y coordinate of (x4, y4).
