Function Prototypes

Description

The following are all the function prototypes used in Dynamsoft Barcode Reader JavaScript SDK.


OnInitSuccess()

Parameters Type Description
none N/A N/A


OnInitFailure(errorCode, errorString);

Parameters Type Description
errorCode Int Gets the error code.
errorString String Gets the error string.


OnBarcodeReadSuccess(userData, result)

Parameters Type Description
userData Object Custom user data, such as image index, image url, etc.
result ReadResult Returns all detected barcode results including format, contents, boundary, angle, etc.


OnBarcodeReadFailure(userData, errorCode, errorString);

Parameters Type Description
userData Object Custom user data, such as image index, image url, etc.
errorCode Int Gets the error code.
errorString String Gets the error string.
