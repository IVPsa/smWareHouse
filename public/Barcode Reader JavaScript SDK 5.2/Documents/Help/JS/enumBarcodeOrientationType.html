EnumBarcodeOrientationType Enumeration

Description

An enumeration that represents the orientation type of the barcodes.

  • Member Description
    EBOT_Horizontal Barcodes with a horizontal orientation will be searched.
    EBOT_Vertical Barcodes with a vertical orientation will be searched.

Remarks

If you set this property, the library will look for barcodes that are skewed from the specified orientation by as much as ±45 degrees.

By default, it will search for barcodes at any angle. To improve recognition and speed performance, please set this property correctly.
