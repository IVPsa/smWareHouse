readBinary Method

Description

Reads barcodes from a blob image synchronously. It supports jpg, png, tif, gif and bmp formats. This method is valid only in IE 10-11, Edge, Chrome v27+ and Firefox v27+.

Syntax

.readBinary(Blob blobImage);

Parameters

blobImage
The binary array of the barcode image.

Return Value

ReadResult

When null is returned, check getErrorCode or getErrorString for error information.

Example

// HTML:
<input type="file" id="file"/>

// JavaScript:
var upLoadFile = document.getElementById("file").files[0];

var count = result.getCount();

        strMsg +="Barcode Value: " + result.get(i).text + "\n";
    }
    alert(strMsg);
}
else {
    alert("No barcode(s) found.");
}
