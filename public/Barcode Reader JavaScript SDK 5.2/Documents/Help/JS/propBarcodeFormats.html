barcodeFormats Property

Description

Gets or sets what types/formats of barcode are read. All other types/formats are ignored.

Syntax

.barcodeFormats = ...;

Data type

Int

Default Value

dynamsoft.dbrEnv.EnumBarcodeFormat.EBF_All

Example

var dbrObject;
dynamsoft.dbrEnv.init(function () { dbrObject = new dynamsoft.dbrEnv.BarcodeReader(); }, function (errCode, errMsg) { alert(errMsg); });

dbrObject.barcodeFormats = dynamsoft.dbrEnv.EnumBarcodeFormat.EBF_CODE_39 | dynamsoft.dbrEnv.EnumBarcodeFormat.EBF_CODE_93;
