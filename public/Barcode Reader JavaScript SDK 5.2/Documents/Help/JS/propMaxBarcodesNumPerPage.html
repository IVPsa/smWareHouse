maxBarcodesNumPerPage Property

Description

Gets or sets the maximum number of barcodes to read per page.

Syntax

.maxBarcodesNumPerPage = ...;

Data type

Int

Default Value

0x7fffffff

Example

var dbrObject;
dynamsoft.dbrEnv.init(function () { dbrObject = new dynamsoft.dbrEnv.BarcodeReader(); }, function (errCode, errMsg) { alert(errMsg); });
dbrObject.maxBarcodesNumPerPage = 200;
