dynamsoft.dbrEnv.init Method

Description

Initializes Dynamsoft Barcode Reader object.

Syntax

dynamsoft.dbrEnv.init(onInitSuccess, onInitFailure);

Data type

Void

Parameters

function OnInitSuccess
Callback function triggered when the initialization succeeds.
function OnInitFailure
Callback function triggered if the initialization fails.

Remarks

If it is initialized successfully, you can call dynamsoft.dcsEnv.BarcodeReader() to get the Object in the onSuccess function. Otherwise, please check error code and error string in the onFailure function.

Example

var dbrObject; 
function onInitSuccess(){
    dynamsoft.dbrEnv.productKey = "***"; // Please replace "***" with your own product key.
    dbrObject = new dynamsoft.dbrEnv.BarcodeReader();
}

function onInitFailure(errorCode, errorString){
    alert("Init failed: " + errorCode + "; " + errorString);
}

dynamsoft.dbrEnv.init(onInitSuccess,onInitFailure);
