addPages Method

Description

Adds page indices of a file (.tiff) to the selection for the barcode search.

Syntax

.addPages(int[] indices);

Parameters

indices
Specifies the indices of images in buffer. The index is 0-based.

Remarks

It works for TIFF files only.

By default, it will search for barcodes in all pages of a file. You can speed up the recognition process by specifying page number(s) if you know barcodes are in which page(s).

Return value

bool

true indicates success. false indicates failure.

When false is returned, check getErrorCode or getErrorString for error information.

Example

var dbrObject;
dynamsoft.dbrEnv.init(function () { dbrObject = new dynamsoft.dbrEnv.BarcodeReader(); }, function (errCode, errMsg) { alert(errMsg); });

dbrObject.clearAllPages();
dbrObject.addPages([0,2,3]);
