EnumBarcodeFormat Enumeration

Description

An enumeration that describes all supported barcode formats.

  • Member Value Barcode Type
    EBF_All 0 All supported formats
    EBF_OneD 1023 One-D barcode
    EBF_CODE_39 1 Code 39
    EBF_CODE_128 2 Code 128
    EBF_CODE_93 4 Code 93
    EBF_CODABAR 8 Codabar
    EBF_ITF 16 Interleaved 2 of 5
    EBF_EAN_13 32 EAN-13
    EBF_EAN_8 64 EAN-8
    EBF_UPC_A 128 UPC-A
    EBF_UPC_E 256 UPC-E
    EBF_INDUSTRIAL_25 512 Industrial 2 of 5
    EBF_PDF417 33554432 PDF417
    EBF_QR_CODE 67108864 QRCode
    EBF_DATAMATRIX 134217728 DATAMATRIX

Remarks

Barcode formats can be combined like this:
dynamsoft.dbrEnv.EnumBarcodeFormat.EBF_CODE_39 | dynamsoft.dbrEnv.EnumBarcodeFormat.EBF_CODE_93
