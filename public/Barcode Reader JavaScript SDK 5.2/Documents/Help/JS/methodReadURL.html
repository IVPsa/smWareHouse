readURL Method

Description

Reads barcodes from an image URL synchronously. It supports jpg, png, tif, gif and bmp formats.

Syntax

.readURL(String imageUrl);

Parameters

imageUrl
The URL of the barcode image. For example: 
http://demo1.dynamsoft.com/dbr/code128.png
https://demo1.dynamsoft.com/dbr/code128.png
dwt://dwt_trial_13000404/img?id=933002725&index=0&t=1488364093533
dcs://dcs_trial_5200408/img?id=933002725&index=0&t=1488364093533

Return Value

ReadResult

When null is returned, check getErrorCode or getErrorString for error information.

Remarks

If the URL you specified is from a different domain/port, then you will encounter Cross-Origin Resource Sharing (CORS) issue.

For example, if you are on the website http://demo1.dynamsoft.com and are trying to access the image ( https://www.dynamsoft.com/demo/DWT/Images/code-39.png ) which is on a different domain, you will get the following error in the console: 

XMLHttpRequest cannot load ***. No 'Access-Control-Allow-Origin' header is present on the requested resource.

In order to allow cross-domain requests, please refer to this KB article.

Example

var result = dbrObject.readURL("http://demo1.dynamsoft.com/dbr/code128.png");

// Show barcode results
var count = result.getCount();
var strMsg = "";
if(count > 0) {
    for (var i = 0; i < count; i++){
        strMsg +="Index: " + i + "\n";
        strMsg +="Barcode Type: " + result.get(i).formatString + "\n";
        strMsg +="Barcode Value: " + result.get(i).text + "\n";
    }
    alert(strMsg);
}
else {
    alert("No barcode(s) found.");
}
