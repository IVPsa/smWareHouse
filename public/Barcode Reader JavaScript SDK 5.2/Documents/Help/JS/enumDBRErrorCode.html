EnumDBRErrorCode Enumeration

Description

An enumeration that describes all the errors defined in Dynamsoft Barcode Reader.

Error Code Constant Error String
0 OK Successful.
-1001 ParameterNumberUnmatched One or more parameters are missing.
-1002 TypeNotValid The type of parameter [parameter name] is invalid.
-1003 ValueOutOfRange The value of parameter [parameter name] is out of range.
-1008 ValueNotValid The value of parameter [parameter name] is invalid.
-1012 HttpError The server returned the error code [HTTP_status_codes].
-1015 LicenseDomainInvalid The domain of your current site does not match the domain bound in the current product key, please contact the site administrator.
-1028 ConversionFailed The conversion to base64 string failed.
-1029 JsonParseFailed Failed to parse json string.
-4001 DownloadDBRModuleFailed Failed to download the DBR module. Please check the network connection and try again later.
-4002 SendDBRModuleFailed Failed to send the DBR module to Dynamsoft Service. Please check the network connection and try again later.
-4003 ConnectDBRModuleFailed Failed to connect to the DBR module. Please check the network connection and try again later.
-10000 Unknown Unknown error.
-10001 NoMemory Not enough memory to perform the operation.
-10002 NullReference Null pointer.
-10004 LicenseExpired The license expired.
-10006 FileTypeNotSupported The file type is not supported.
-10008 IndexInvalid The index is invalid.
-10009 BarcodeFormatInvalid The barcode format is invalid.
-10010 CustomedRegionInvalid Barcode custom region to decode is invalid.
-10011 MaxBarcodeNumberInvalid The maximum barcode number is invalid.
-10012 ImageReadFailed Failed to read the image.
-10013 TiffReadFailed Failed to read the TIFF image.
-10016 LicenseQRInvalid You do not have a valid QR Barcode license.
-10017 License1DInvalid You do not have a valid 1D Barcode license.
-10018 DIBBufferInvalid The DIB (Device-independent bitmaps) buffer is invalid.
-10019 LicensePDF417Invalid You do not have a valid PDF417 barcode license.
-10020 LicenseDataMatrixInvalid You do not have a valid DATAMATRIX barcode license.
-10023 PageNumberInvalid The page number is invalid.
-10026 RecognitionTimeout Recognition timeout.
-20000 ServiceInternalError ([service internal code]) [error string returned from service]
