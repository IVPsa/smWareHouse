barcodeColorMode Property

Description

Gets or sets the ink color for barcodes search.

Syntax

.barcodeColorMode = ...;

Data type

EnumBarcodeColorMode

Default Value

dynamsoft.dbrEnv.EnumBarcodeColorMode.EBCM_DarkOnLight

Example

var dbrObject;
dynamsoft.dbrEnv.init(function () { dbrObject = new dynamsoft.dbrEnv.BarcodeReader(); }, function (errCode, errMsg) { alert(errMsg); });

dbrObject.barcodeColorMode = dynamsoft.dbrEnv.EnumBarcodeColorMode.EBCM_DarkAndLight;
