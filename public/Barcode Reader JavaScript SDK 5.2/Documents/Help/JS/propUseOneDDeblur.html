useOneDDeblur Property

Description

Sets whether to deblur 1D barcodes before scanning.

Syntax

.useOneDDeblur = ...;

Data type

Boolean

Default Value

true

Remarks

It is only valid for 1D barcode types.

Example

var dbrObject;
dynamsoft.dbrEnv.init(function () { dbrObject = new dynamsoft.dbrEnv.BarcodeReader(); }, function (errCode, errMsg) { alert(errMsg); });

dbrObject.useOneDDeblur = true;
