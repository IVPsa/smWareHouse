imageCaptureDevice Property

Description

Gets or sets the type of devices you used to capture barcode images.

Syntax

.imageCaptureDevice = ...;

Data type

EnumImageCaptureDevice

Default Value

dynamsoft.dbrEnv.EnumImageCaptureDevice.EICD_Unknown

Example

var dbrObject;
dynamsoft.dbrEnv.init(function () { dbrObject = new dynamsoft.dbrEnv.BarcodeReader(); }, function (errCode, errMsg) { alert(errMsg); });

dbrObject.imageCaptureDevice = dynamsoft.dbrEnv.EnumImageCaptureDevice.EICD_Unknown;
