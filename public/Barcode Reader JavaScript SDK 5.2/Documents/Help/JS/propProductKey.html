dynamsoft.dbrEnv.productKey Property

Description

Sets or gets the product key string for Dynamsoft Barcode Reader.

Data Type

String

Syntax

dynamsoft.dbrEnv.productKey = "...";

Remarks

If you have more than one product key, please use a semicolon (;) to separate them. For example, 

// Set the license in the dynamsoft.barcodereader.config.js file:
dynamsoft.dbrEnv.productKey = "key1;key2";

If you deploy your application to a web server and set both trial and full product keys, the library will consider you are in a production environment and all trial keys will be ignored.
