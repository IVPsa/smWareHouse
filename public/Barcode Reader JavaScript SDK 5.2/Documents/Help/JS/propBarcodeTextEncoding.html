barcodeTextEncoding Property

Description

Gets or sets barcode text encoding mode.

Syntax

.barcodeTextEncoding = ...;

Data type

EnumBarcodeTextEncoding

Default Value

dynamsoft.dbrEnv.EnumBarcodeTextEncoding.EBTE_Default

Example

var dbrObject;
dynamsoft.dbrEnv.init(function () { dbrObject = new dynamsoft.dbrEnv.BarcodeReader(); }, function (errCode, errMsg) { alert(errMsg); });

dbrObject.barcodeTextEncoding = dynamsoft.dbrEnv.EnumBarcodeTextEncoding.EBTE_Default;
