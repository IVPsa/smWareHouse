EnumBarcodeColorMode Enumeration

Description

An enumeration that describes all ink colors for barcodes search.

  • Member Value Description
    EBCM_DarkOnLight
    (Default)    		 0 Barcodes are printed with dark color (usually in black) on a light (usually in white) background.
    EBCM_LightOnDark 1 Barcodes are printed with light color (usually in white) on a dark (usually in black) background.
    EBCM_DarkAndLight 2 Both of the above

Remarks

When you choose EBCM_LightOnDark mode, the library will invert colors before decoding.
